12) To make a new Pull Request:
0)Copy New pr.bat to newmod main folder (or else it will fail to delete the help folder).
a) Run new PR.bat
b) Open gitbash> git checkout -b vanilla (in case it fails to checkout)
c) git checkout -b <branch name here>
d) Push commits> push to origin> Done. 

Help:
for debug, just run debug compile.bat

manual:
Note: Always name the main mod folder as "newmod" in order to make the script work.
Debug
cd /d C:\newmod 
meson build-debug 
cd build-debug 
ninja

for static
cd /d C:\newmod 
meson -Dbuildtype=release -Dstatic=prebuilt -Db_vscrt=static_from_buildtype build-release-static
cd build-release-static
ninja
